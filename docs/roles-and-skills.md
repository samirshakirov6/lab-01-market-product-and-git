## Roles and responsibilities

1 Back-end engineer –works with databases and writes server logic.  
2 Front-end engineer – creates the user interface and ensures it works.  
3 QA engineer – tests features and reports bugs.  
4 Data engineer – processes and manages logistics-related data.  
5 DevOps engineer – deploys services and monitors servers.

## Common skills across roles

2 Databases: PostgreSQL, MySQL  
3 Testing: Unit tests 
4 Deployment: Git, Docker
